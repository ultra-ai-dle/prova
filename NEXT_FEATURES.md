# 다음 기능 추가 목록

> 생성일: 2026-04-12 | 기준 브랜치: mindaaaa/guide-tour-ui

## 현재 지원 현황

### 시각화 뷰
| 패널 | 지원 자료구조 | 비고 |
|------|-------------|------|
| GridLinearPanel | 1D 배열, 2D 그리드(DP 테이블, 미로) | two-pointer/sliding-window 피벗 지원 |
| GraphPanel | 인접리스트/맵, 트리, 방향/무방향 그래프 | D3 force layout, GRID↔GRAPH 토글 |
| ThreeDVolumePanel | 3D 배열, 비트마스크 확장 | Three.js 슬라이스 뷰 |
| specialViews (8종) | Heap, Queue, Stack, Deque, UnionFind, Visited, Distance, ParentTree | GraphPanel 내부 자동 전환 |

### AI 분류 능력
- **전략**: GRID / LINEAR / GRID_LINEAR / GRAPH
- **감지 가능 알고리즘**: 정렬(8종), 그래프 탐색(BFS/DFS/Dijkstra/Bellman-Ford/Floyd), MST(Kruskal/Prim), DP(피보나치/배낭/LIS/LCS/편집거리/비트마스크), Two-pointer, Sliding Window, 이진탐색, KMP, Rabin-Karp, Union-Find 등
- **자동 enrichment**: deque/heap/stack 패턴, pivot 감지, graph mode 추론

### UX
- 3-column 리사이즈 가능 레이아웃 (100vh 고정)
- 코드 에디터 (Python/JS, 문법 강조, Tab/Wrap 토글)
- 재생 컨트롤 (Prev/Next/Play/Speed ×0.5~×100/Slider/Jump-to-error)
- 키보드 내비게이션 (←→, Space, Home/End)
- 7단계 가이드 투어 (첫 방문 자동 시작)
- 에러 시각화 (빨간 라인 강조, 배너, 타임라인 점프)
- localStorage 코드/언어/입력 저장

## 🔴 High Priority

| 기능 | 분류 | 이유 | 예상 난이도 |
|------|------|------|------------|
| AI 설명 패널 UI | UX | SSE로 받는 explanation이 화면에 표시되지 않음. 핵심 가치인 "자연어 해설" 미노출 | ★★☆ |
| 트리 전용 시각화 | 시각화 | 트리는 GRAPH로 렌더링되지만 계층 레이아웃이 없어 가독성 떨어짐 (이진트리, BST, 세그먼트트리 등) | ★★★ |
| Java/C++ 언어 지원 | 실행 | 드롭다운에 "준비중"으로 존재. 알고리즘 학습자 대다수가 사용하는 언어 | ★★★★ |
| DP 테이블 전이 화살표 | 시각화 | DP에서 현재 셀이 어떤 셀들로부터 값을 가져오는지 시각적 연결 없음 | ★★☆ |
| 코드-시각화 동기화 강조 | UX | 현재 실행 줄은 강조되지만, 시각화 요소와 코드 변수의 연결이 명시적이지 않음 | ★★☆ |
