<blockquote>브라우저에서 Python / JavaScript / Java 코드를 돌리면, AI가 알고리즘을 분류하고<br/>실행 흐름을 스텝별 시각화와 자연어 해설로 변환합니다.</blockquote>

### 1. 코드 실행 — 브라우저에서 바로

Python(Pyodide WASM)과 JavaScript(Acorn AST)를 설치 없이 브라우저에서 실행합니다.

<!-- ![코드 실행 데모](demo/01-execution.gif) -->


<br clear="both" />

### 2. AI 알고리즘 분류

코드를 분석해 알고리즘 종류를 자동 판별하고, 최적의 시각화 전략을 결정합니다.

<!-- ![알고리즘 분류 데모](demo/02-classification.gif) -->


<br clear="both" />

### 3. 스텝별 자연어 해설

각 실행 단계마다 AI가 무슨 일이 일어나는지 자연어로 설명합니다. SSE 스트리밍으로 실시간 표시됩니다.

<!-- ![자연어 해설 데모](demo/03-explanation.gif) -->


<br clear="both" />

### 4. 다양한 시각화 모드

Grid / Graph / Linear / 3D Volume 등 알고리즘 특성에 맞는 시각화를 자동 선택합니다.

<!-- ![시각화 모드 데모](demo/04-visualization.gif) -->


<br clear="both" />

### 5. 자료구조 전용 뷰

힙, 큐, 스택, 데크, Union-Find 등 자료구조별 특화된 시각화를 제공합니다.

<!-- ![자료구조 뷰 데모](demo/05-data-structures.gif) -->


<br clear="both" />

### 6. 포인터 & 피벗 자동 감지

Two-pointer, 퀵소트 피벗 등을 코드 맥락에서 자동으로 감지하고 시각화에 반영합니다.

<!-- ![포인터 감지 데모](demo/06-pointers.gif) -->

<div align="center">

**TRACE** &nbsp; **AI** &nbsp; **RENDER**

<p>
<sup><strong>TRACE</strong> - Web Worker에서 코드를 실행하고 줄 단위 <code>rawTrace</code>를 수집합니다.</sup><br />
<sup><strong>AI</strong> - <code>analyze</code>로 알고리즘과 변수를 분류하고, <code>explain</code>으로 스텝별 해설을 SSE로 붙입니다.</sup><br />
<sup><strong>RENDER</strong> - <code>mergedTrace</code>를 기준으로 Grid / Graph / Linear 등 시각화 패널을 그립니다.</sup>
</p>

### 설치 및 실행

```sh
# 저장소 클론
git clone https://github.com/ultra-ai-dle/prova.git

# 프로젝트 루트로 이동
cd prova

# 의존성 설치
npm install

# 환경 변수 파일 생성 (아래「환경변수」참고)
cp env.example .env.local

# 개발 서버 실행 — http://localhost:3000
npm run dev
```

<br />

### 환경변수

```
# ─── AI API Keys (사용할 provider 만 설정)
GEMINI_API_KEY=
OPENAI_API_KEY=
GROQ_API_KEY=
ANTHROPIC_API_KEY=
OPENROUTER_API_KEY=

# ─── 메인 / 폴백 모델
AI_PROVIDER=gemini
AI_MODEL=gemini-2.5-flash
FALLBACK_PROVIDER=gemini
FALLBACK_MODEL=gemini-2.5-flash-lite

# ─── Java 원격 실행 (Java 언어 사용 시 필수 · 토큰은 upstream 정책에 따라)
JAVA_EXECUTION_SERVICE_URL=
JAVA_EXECUTION_SERVICE_TOKEN=
```
