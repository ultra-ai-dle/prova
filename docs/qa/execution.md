# QA: Execution Engine

> 마지막 업데이트: 2026-04-11
> 대상 파일: src/features/execution/runtime.ts

## 정상 동작
- [ ] **Python 실행** — Python 코드를 입력하고 실행하면 → 하단 패널이 시각화 탭으로 자동 전환되고 스텝 슬라이더가 활성화된다
- [ ] **JS 실행** — JavaScript 코드를 입력하고 실행하면 → Python과 동일하게 시각화 패널이 전환되고 스텝 탐색이 가능하다
- [ ] **stdin 입력** — Python에서 stdin이 필요한 코드를 실행하면 → stdin 입력 영역에 값을 넣은 뒤 정상 실행된다
- [ ] **JS stdin 불필요** — JavaScript 코드는 stdin을 비워두어도 → 실행이 정상적으로 진행된다
- [ ] **재실행** — 실행 완료 후 코드를 수정하고 다시 실행하면 → 이전 시각화가 초기화되고 슬라이더가 0으로 리셋된다
- [ ] **언어 자동 감지** — Python 키워드가 포함된 코드를 JS 모드에서 실행하면 → "코드 패턴을 감지해 Python으로 전환했습니다" 토스트가 표시된다
- [ ] **코드 영속성** — 코드를 입력하고 페이지를 새로고침하면 → 이전에 입력한 코드, stdin, 언어 선택이 복원된다
- [ ] **디버그 모드 진입** — 실행이 시작되면 → 코드 에디터가 읽기 전용이 되고 편집할 수 없다
- [ ] **디버그 모드 해제** — 연필 아이콘을 클릭하면 → 에디터가 다시 편집 가능해진다

## 엣지 케이스
- [ ] **빈 코드 제출** — 코드 없이 실행 버튼을 누르면 → "코드를 입력한 후 디버깅을 시작하세요" 토스트가 표시되고 실행되지 않는다
- [ ] **Python stdin 미입력** — Python 코드에서 stdin을 비워두고 실행하면 → "예시 입력(stdin)을 입력한 후 디버깅을 시작하세요" 토스트가 표시된다
- [ ] **대량 스텝** — 10,000 step을 초과하는 코드를 실행하면 → step limit 경고가 표시되고 수집이 중단되며, 수집된 범위까지 시각화된다
- [ ] **순환 참조** — 순환 참조가 있는 객체를 사용하면 → 변수 패널에 `<circular>`로 표시된다
- [ ] **NaN/Infinity 값** — NaN이나 Infinity 값이 발생하면 → 변수 패널에 문자열 "NaN", "Infinity"로 표시된다
- [ ] **대용량 컬렉션** — 30개 이상의 항목을 가진 리스트를 사용하면 → 초과분은 "...(+N)"으로 잘려서 표시된다
- [ ] **깊은 중첩 객체** — 3단계 이상 중첩된 객체는 → 3단계까지만 표시되고 이후는 잘린다
- [ ] **SystemExit** — Python 코드에서 `sys.exit()`을 호출하면 → 에러가 아닌 정상 종료로 처리된다

## 에러 케이스
- [ ] **문법 에러** — 문법 에러가 있는 코드를 실행하면 → 에디터에 에러 표시가 나타나고 하단에 에러 메시지가 출력된다
- [ ] **무한루프** — 무한루프 코드를 실행하면 → 120초 후 "실행 시간이 너무 길어 안전을 위해 중단하고 환경을 재설정합니다" 토스트가 표시된다
- [ ] **타임아웃 후 복구** — 타임아웃이 발생한 후 → 약 1초 뒤 "환경 준비 완료" 토스트가 표시되고 재실행이 가능해진다
- [ ] **Worker 크래시** — Worker가 예기치 않게 종료되면 → 빨간색 배너에 "Python 환경 초기화에 실패했습니다. 페이지를 새로고침해 주세요"가 표시된다
- [ ] **런타임 에러** — 코드 실행 중 에러가 발생하면 → 에러가 발생한 라인이 빨간색으로 하이라이트되고 출력 패널에 Traceback이 표시된다

## UI 상태
- [ ] **실행 중 표시** — 코드 실행이 시작되면 → 상단에 진행 바가 나타나고 실행 버튼이 "디버깅 중..."으로 변경되어 비활성화된다
- [ ] **Python 로딩** — 페이지 첫 로드 시 → 실행 버튼이 "Python 준비 중..."으로 표시되고 로딩이 완료되면 "▶ 디버깅 시작"으로 변경된다
- [ ] **실행 버튼 레이블** — 실행 완료 후 → 실행 버튼이 "▶ 디버깅 다시 실행"으로 변경된다
- [ ] **언어 선택** — Python, JavaScript, Java는 선택 가능하고, C++은 "준비중"으로 비활성화되어 있다
- [ ] **탭 크기 전환** — Tab 2/Tab 4 토글 버튼을 클릭하면 → 기존 코드의 들여쓰기가 해당 크기로 변환된다
- [ ] **코드 복사** — 복사 버튼을 클릭하면 → 체크마크 아이콘이 1.5초간 표시된 후 원래 아이콘으로 돌아간다
