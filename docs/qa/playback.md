# QA: Playback

> 마지막 업데이트: 2026-04-12
> 대상 파일: src/features/playback/TimelineControls.tsx, src/store/useProvaStore.ts, app/page.tsx

## 정상 동작
- [ ] **스텝 앞으로 이동** — Next 버튼을 클릭하면 → 다음 스텝으로 이동하고 시각화가 갱신된다
- [ ] **스텝 뒤로 이동** — Prev 버튼을 클릭하면 → 이전 스텝으로 이동하고 시각화가 이전 상태로 복원된다
- [ ] **자동 재생** — Play 버튼을 클릭하면 → 스텝이 자동으로 하나씩 앞으로 진행된다
- [ ] **재생 일시정지** — 재생 중 Pause 버튼을 클릭하면 → 현재 스텝에서 멈추고 수동 탐색이 가능해진다
- [ ] **재생 속도 조절** — 속도 드롭다운에서 x0.5/x1/x1.5/x2를 선택하면 → 자동 재생 간격이 해당 속도로 변경된다
- [ ] **타임라인 점 클릭** — 타임라인의 특정 스텝 점을 클릭하면 → 해당 스텝으로 즉시 이동한다
- [ ] **에러 스텝 점프** — 에러가 있는 트레이스에서 에러 점프 버튼을 클릭하면 → 첫 번째 에러 스텝으로 이동한다
- [ ] **키보드 좌우 화살표** — 좌/우 방향키를 누르면 → 이전/다음 스텝으로 이동한다
- [ ] **키보드 Space** — Space 키를 누르면 → 재생/일시정지가 토글된다
- [ ] **키보드 Home/End** — Home 키를 누르면 첫 스텝, End 키를 누르면 마지막 스텝으로 이동한다

## 엣지 케이스
- [ ] **첫 스텝에서 Prev** — 스텝 0에서 Prev 버튼은 → 비활성화되어 클릭할 수 없다
- [ ] **마지막 스텝에서 Next** — 마지막 스텝에서 Next 버튼은 → 비활성화되어 클릭할 수 없다
- [ ] **마지막 스텝에서 재생** — 마지막 스텝에서 Play를 누르면 → 다음 스텝이 없으므로 즉시 재생이 멈춘다
- [ ] **단일 스텝 트레이스** — 1개 스텝만 있는 트레이스에서 → Prev/Next 모두 비활성화되고 타임라인에 점 1개만 표시된다
- [ ] **빈 트레이스** — 실행 결과가 없으면 → "타임라인 없음 — 코드를 실행하세요" 메시지가 표시되고 모든 버튼이 비활성화된다
- [ ] **입력 필드에서 키보드** — 텍스트 입력 영역에 포커스된 상태에서 화살표/Space를 누르면 → 재생 컨트롤이 동작하지 않고 입력이 정상 처리된다

## 에러 케이스
- [ ] **재생 중 에러 스텝 도달** — 자동 재생 중 런타임 에러 스텝에 도달하면 → 재생이 자동으로 멈추고 에러 스텝 모드로 전환된다

## UI 상태
- [ ] **실행 중 비활성화** — 코드가 실행 중이면 → 모든 재생 컨트롤(버튼, 속도, 타임라인)이 비활성화된다
- [ ] **타임라인 색상 구분** — 타임라인 점은 → 일반 스텝(회색), 루프 스텝(파란색), 분기 스텝(보라색 다이아몬드), 에러 스텝(빨간색)으로 구분된다
- [ ] **현재 스텝 강조** — 현재 스텝의 타임라인 점에 → 링과 하이라이트가 표시된다
